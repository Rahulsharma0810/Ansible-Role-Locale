Ansible-Role-Locale
=========

Super Simple Role that just Setup Locales on linux sytems.

Role Variables
--------------
```
LANG: LANG=en_US.utf-8
LC_CTYPE: LC_CTYPE=en_US.UTF-8
LC_ALL: LC_ALL=en_US.utf-8
```
Default English US English is set.


Example Playbook
----------------

    - hosts: YOUR-HOST-OR-GROUP
      roles:
    	- Ansible-Role-Locale
